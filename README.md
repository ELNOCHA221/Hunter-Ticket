# Hunter-Ticket
Extensión de Chrome (Manifest V3) que automatiza el reclamo de tickets en servidores de Discord. Detecta canales de tickets, presiona el botón de reclamar antes que nadie, envía respuestas automáticas, maneja modales, filtra por servidor con whitelist, y notifica vía webhook. Incluye overlay en tiempo real con estadísticas. Hecho por BOTX.
